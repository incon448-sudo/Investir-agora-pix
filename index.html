⚡ Como usar o painel

Navegue pelas abas acima para gerenciar saques, depósitos e usuários.

1 Aba "Saques" → Aprovar/Rejeitar saques pendentes
2 Aba "Depósitos" → Confirmar pagamentos recebidos
3 Aba "Usuários" → Ver todos os usuários cadastrados

💡 Gerenciamento de Saques

Aqui você aprova ou rejeita os pedidos de saque dos usuários. Ao aprovar, o saque é marcado como pago. Ao rejeitar, o valor volta para o saldo do usuário automaticamente.

1 Veja os saques pendentes na lista abaixo
2 Clique em "Aprovar" para confirmar o pagamento Pix
3 Clique em "Rejeitar" para devolver o valor ao usuário

Aqui você confirma os pagamentos que os usuários fizeram via Pix. Ao aprovar, o valor é creditado no saldo deles. Verifique se o pagamento realmente caiu antes de aprovar.

1 Verifique se o pagamento caiu na conta
2 Clique em "Aprovar" para creditar o valor
3 Clique em "Rejeitar" se o pagamento não foi identificado
